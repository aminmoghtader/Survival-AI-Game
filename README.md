# Survival - 2D Space Shooter with AI

Survival is a 2D space shooter game developed in Python using **Pygame**, where you battle alien enemies in an endless space environment. The game integrates a basic AI to control the movement of the player's spaceship, with plans for further enhancements.

## Features

- 2D space shooting gameplay with alien enemies.
- Basic AI-controlled spaceship movement.
- Modular AI system allowing future upgrades.
- Fully developed in Python using Pygame.

Usage
Arrow keys / WASD to control the ship manually.
The AI can be toggled to take control of the ship's movement.
Score points by destroying alien ships and surviving longer.
